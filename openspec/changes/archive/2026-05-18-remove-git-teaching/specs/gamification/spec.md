## REMOVED Requirements

### Requirement: Repository MUST include a PROGRESS.md tracker
**Reason**: Module-based learning with PROGRESS.md tracking has been removed. The teach-anything skill does not use pre-defined modules or progress tracking.
**Migration**: Progress is tracked conversationally within the AI agent session.

### Requirement: Each module completion SHALL claim a badge
**Reason**: The badge.sh script and module-completion badge system have been removed.
**Migration**: No migration needed.

### Requirement: Badge script SHALL verify before claiming
**Reason**: Badge.sh no longer exists.
**Migration**: No migration needed.

### Requirement: Full completion SHALL trigger a "graduation" ceremony
**Reason**: Module-based graduation ceremony has been removed.
**Migration**: No migration needed.
