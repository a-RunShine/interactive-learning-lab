# 模块 0：命令行生存手册 — 知识总结

## 核心思维

命令行是用文字指令代替鼠标点按来操作电脑。Shell 是翻译官，Bash 是其中一种（Git Bash = MINGW64）。

## 命令清单

| 命令 | 作用 | 常用变体 |
|------|------|---------|
| `pwd` | 显示当前路径 | — |
| `ls` | 列出文件 | `-l` 详细、`-a` 显示隐藏、`-R` 递归 |
| `cd` | 切换目录 | `cd ..` 上级、`cd ~` 回家、`cd -` 回上一步 |
| `mkdir` | 创建目录 | `mkdir -p a/b/c` 递归创建 |
| `touch` | 创建空文件 | 已存在则更新时间戳 |
| `echo` | 打印内容 | `>` 覆盖写入、`>>` 追加写入 |
| `cat` | 查看文件内容 | — |
| `rm` | 删除 | `rm 文件`、`rm -r 目录`、`rm -rf`（慎用） |
| `mv` | 移动/重命名 | — |

## 通配符

- `*` 匹配任意字符：`rm *.class`、`ls *.java`
- `?` 匹配单个字符
- `[abc]` 匹配集合中任一字符

## 终端生存技能

- **Tab 补全** — 敲前几个字母按 Tab
- **↑ 历史命令** — 找回上次敲的命令
- **Ctrl+C** — 中断卡住的程序
- **Ctrl+S** / **Ctrl+Q** — 冻结 / 解冻屏幕

## Vim 三进三出

```
vim 文件名    → 进入
按 i          → 进入插入模式，开始打字
Esc           → 退出插入模式
:wq 回车      → 保存退出
:q! 回车      → 不保存强制退出
```

## 路径认知

- `.` = 当前目录，`..` = 上级目录
- `.` 开头的文件默认隐藏，`ls -a` 才能看到
- 绝对路径从 `/` 或 `C:/` 开始，相对路径从当前位置开始
